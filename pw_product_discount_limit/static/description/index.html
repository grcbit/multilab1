Product Discount Limit

This module is allow to set discount limit on product and product category



App Features
  • Set discount limit on product
  • Set discount limit on product category
  • Restrict to apply discount more than discount limit
  • Discount restriction on sale order
  • Discount restriction on invoice/bills
  • Compatible with community and enterprise
Product Discount Limit

You can set discount limit on product category, when user apply discount in sale order/invoice/bills then system will restrict to apply discount on these category of products if discount more than limit


You can set discount limit on product (system will first check discount limit on product)


You can set discount limit on product variants

When user apply discount more than discount limit on sale order, system will restrict to apply discount (if you have set discount limit on product and category, then system will first check discount limit on product), and system will auto set descount to zero


Also same restriction on customer invoice/vendor bills

System will allow to apply discount more than limit of these group of users
